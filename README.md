# Pr-cticas-sobre-Arreglos-Bidimensionales

## 1: Sistema de Promedio de Estudiantes

### Descripción

Aplicación diseñada para gestionar y calcular promedios académicos de estudiantes en múltiples asignaturas. Permite registrar calificaciones de 4 estudiantes en 3 asignaturas diferentes y calcular tanto promedios individuales como promedios por materia.

### Características Principales

- **Gestión de Asignaturas**: Permite ingresar nombres personalizados para 3 asignaturas
- **Registro de Estudiantes**: Captura nombres de 4 estudiantes
- **Ingreso de Calificaciones**: Sistema de entrada con validación de datos numéricos
- **Cálculo Automático**: Procesa automáticamente promedios individuales y por asignatura
- **Manejo de Errores**: Validación mediante try-catch para entradas incorrectas

### Funcionalidades

1. **Promedio por Estudiante**: Calcula el rendimiento promedio de cada estudiante considerando sus 3 calificaciones
2. **Promedio por Asignatura**: Determina el rendimiento promedio grupal en cada materia
3. **Validación de Datos**: Asegura que solo se ingresen valores numéricos válidos
## 2: Sistema de Procesamiento de Matrices

### Descripción

Aplicación versátil que permite trabajar con matrices de dimensiones dinámicas definidas por el usuario. Identifica automáticamente los valores máximo y mínimo dentro de la matriz junto con sus posiciones exactas.

### Características Principales

- **Dimensiones Dinámicas**: El usuario define el tamaño de la matriz (filas y columnas)
- **Ingreso Manual**: Captura de valores elemento por elemento
- **Visualización Tabular**: Muestra la matriz en formato organizado y legible
- **Análisis Automático**: Identifica valores extremos y sus ubicaciones
- **Validación Robusta**: Manejo de excepciones para garantizar datos válidos

### Funcionalidades

1. **Creación de Matriz Personalizada**: Permite definir matrices de cualquier tamaño
2. **Búsqueda de Valor Máximo**: Identifica el elemento mayor y su posición [fila, columna]
3. **Búsqueda de Valor Mínimo**: Localiza el elemento menor y su posición [fila, columna]
4. **Visualización Clara**: Presenta los datos en formato tabular profesional
